# Redefinir PIN de acesso

PINs não podem ser redefinidos pelo próprio usuário pelo sistema — por enquanto, isso precisa ser feito pelo administrador do sistema.

---

## Se você esqueceu seu PIN

Entre em contato com o administrador do sistema (secretaria da ICM Newark) e informe:
- Seu nome completo
- Seu perfil de acesso (Atendente, Pastor, Líder de GA, etc.)

O administrador vai redefinir seu PIN e te informar o novo valor.

---

## Se seu PIN está sendo recusado

Antes de pedir redefinição, verifique:

**1. Você está digitando no campo certo?**  
Certifique-se de que o cursor está no campo de PIN antes de digitar.

**2. O Caps Lock está ativado?**  
PINs são numéricos, então Caps Lock não deveria interferir — mas verifique mesmo assim.

**3. Você está usando o PIN correto para o seu perfil?**  
Cada perfil tem um PIN diferente. Se você tem mais de um perfil, use o PIN correspondente ao acesso que você quer.

**4. O sistema está respondendo?**  
Tente recarregar a página. Se continuar recusando, pode ser um problema de conexão.

Se nada disso resolver, entre em contato com o administrador.

---

## Se você é o administrador

Para redefinir o PIN de um usuário:

1. Acesse o painel **Admin**
2. Vá em **Usuários do sistema**
3. Localize o usuário pelo nome ou perfil
4. Clique em **Editar** e atualize o PIN
5. Salve e comunique o novo PIN ao usuário diretamente

> ⚠️ PINs são armazenados no banco de dados. Escolha valores que não sejam óbvios (evite sequências como `1111`, `1234`).
